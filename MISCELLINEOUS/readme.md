## Project Overview
This project is a **simple, responsive website** for Mukangane General Trading, a farm based in Tshiozwi Village, Makhado, South Africa.  
The website includes four pages:

1. **Home** - Welcome page with an overview of the farm.  
2. **About Us** - Information about the farmâ€™s history, mission, and values.  
3. **Services** - List of services offered by the farm, including produce, poultry, livestock, and community training.  
4. **Contact** - Contact information including email, phone, and location.  

The website is styled using an **external CSS file (`style.css`)** to ensure consistency across all pages.
